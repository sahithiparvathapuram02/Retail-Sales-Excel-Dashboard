# Retail-Sales-Excel-Dashboard
Interactive retail sales analytics dashboard built using Microsoft Excel.

**📌PURPOSE**

This project presents an interactive retail sales dashboard built in Microsoft Excel to analyze order performance, revenue trends, product demand, delivery timelines, and customer ratings.

**🗂️ DATA SOURCES**
|Table|Description|
|-----|-----------|
|RETAILSALES_DATA|TX ID,Product,Quantity,Unit Price,Amount,Order Date,Ship Date,Customer Gender,Order,Mode,Rating C	State,County,Days to Deliver,Weeknum,Gender Value|

**🛠️TECH STACK**
* Microsoft Excel

* Pivot Tables & Pivot Charts

* Slicers

* Conditional Formatting

**🔍KEY KPIs**
* Total Orders: 2,400

* Total Quantity: 11,997

* Total Revenue: $649,000

* Average Rating: 4.0

**🧩Analysis Performed**

* Weekly Sales Trend Analysis

* Sales Contribution by Channel & Gender

* Order Quantity Distribution

* Product Performance Analysis

* Geographic Sales Mapping

* Delivery Time Distribution

* Monthly Ratings Analysis

**🔍KEY INSIGHTS AND FINDINGS**

* Week 8 recorded highest revenue

* 2-item orders were most frequent

* Website channel drove maximum sales

* Majority deliveries completed within 2 days
